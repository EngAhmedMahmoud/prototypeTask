#### Login API

---

- REST Name: api/v1/login
- Request Method: POST

- Request parameters:

| name     | description | Mandatory? | default | allowed values | data type |
| -------- | ----------- | ---------- | ------- | -------------- | --------- |
| email    | user email  | Yes        | N/A     | string         | string    |
| password | password    | Yes        | N/A     | string         | string    |
